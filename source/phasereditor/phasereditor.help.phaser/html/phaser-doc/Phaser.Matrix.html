Phaser. Matrix

new Matrix(a, b, c, d, tx, ty)

The Matrix is a 3x3 matrix mostly used for display transforms within the renderer.

It is represented like so:

| a | b | tx |
| c | d | ty |
| 0 | 0 | 1 |
